# Getting Started

## Installation

```bash
pip install scripthut
```

## Configuration

Create a `scripthut.yaml` file in your project directory. See the [example configuration](https://github.com/tlamadon/scripthut/blob/main/scripthut.example.yaml) for all available options.

## Running

```bash
scripthut
```

This starts the web interface on `http://localhost:8000` by default.
